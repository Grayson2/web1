[&lt; All Assignments](../../index.html) # A1: Simple Mockups ## Assignment Description Copy and save out the given mockups in Photoshop. This assignment is intended to give you practice working in Photoshop and to get your coding environment set up to turn in your work. ## Instructions 1. Find the mockups to copy in the [/instructions/examples](examples/) folder 1. Create each mockup page in Photoshop - In Photoshop, create a new document (File > new document) - Use the Document Type: Web, with the preset: `Web (1280,800)` - Save the .psd file - Copy the mockup files changing the names inside to yours. - On the home page, add the 3 card images. (You can find them in the [/instuctions/images](images/) folder) - on the about page, add a photo of yourself. - Do the same for each of the 5 pages 1. Save out each mockup as a flat PNG 1. Upload the images to the `assignment-1/images` folder - once uploaded, you should be able to view all your mockups by opening your `assignment 1/index.html` file. - if there's broken images, check the image's filename. Remember, capitalization matters. 1. Turn in 1. Extra Credit: Take and link screenshots of 2 other example blogs. ## Requirement Checklist: - Can you access the assignment mockups online? `https://{username}.github.io/web1/assignment-1` - Are the images saved as .png not .psd files? - Are there 5 mockups? home, about, coffee, search, and assignments? - Does the content on each mockup generally match the [example images](examples/)? - Spot check the home page: - Is the name on the title your own? - Are all 3 card images present? - Spot check the coffee page: - is there a gray box around the content? - Are the descriptions indented? - Spot check the assignments page: - Are the assignment titles yellow and bold? - Is the list of assignments center aligned - Discuss: What was hard/easy about creating the mockups?
